Brand Identity System · Portfolio Edition

Algonquin College

A comprehensive brand style guide — identity, colour, and typography for Ottawa's largest polytechnic, built to stay consistent across web, merchandise, and print.

Role

Brand identity, logo system, colour & type direction

Deliverables

Logo usage guidelines, colour palette, typography system

Tools

Adobe Illustrator, Figma

Download brand guidelines (PDF) ↓

01 — Overview & logo

Scale first, then symbol

Before drawing a single mark, the brief started with what Algonquin actually is: a 19,000-student polytechnic spread across three campuses. The eagle that follows was built to carry that scale — bold enough for a building facade, restrained enough for a business card.

02 — Colour

A palette built on trust

One dominant blue, one supporting aqua, and a single gold accent held in reserve for calls to action — kept deliberately narrow so the palette reads as institutional, not busy.

03 — Typography

A two-voice type system

An editorial serif for headlines, paired with a functional grotesque for everything else — the same split that carries most well-run institutional identities.
